## 🧱 YOLOV11 / YOLOV12 / YOLOV26 改进项目

基于YOLO最新版本（v11 / v12 / v26）进行系统性结构优化，提供完整可运行代码与改进模块配置文件。  

- 所有改进均采用模块化设计，支持通过 YAML 灵活组合创新  
- 内置多种主流改进方向（注意力机制 / 特征融合 / 轻量化模块等）  
- 支持检测、分割、旋转目标检测等多任务扩展  
- 配备详细说明文档与视频讲解（复杂模块重点解析）  

👉 适合：论文创新 / 消融实验 / 快速复现改进思路

## 🧱 YOLOV8 / YOLOV11 / YOLOV12 / YOLOV26 双Backbone改进项目

针对复杂场景（小目标、遮挡、低对比度）设计的双主干网络结构优化方案。

核心特点：
- 双Backbone提取不同层次特征（语义信息 + 细节信息）  
- 支持多种融合策略（Concat / Attention / 加权融合）  
- 多数据集验证，mAP 提升稳定  

🚀 **扩展能力：**  
该结构不仅支持单模态输入，同时**可直接扩展为双模态结构（RGB + IR / 多源数据）**，实现从双Backbone到多模态融合的平滑过渡。

👉 适合：  
- 拉指标（效果提升明显）  
- 做结构创新论文
