# 双模态目标检测系统

基于 **YOLO 系列** 的双骨干网络实现，支持 RGB + 红外/深度图等多模态输入。

## 🚀 核心特性

- **双骨干架构**: 独立处理两种输入模态（RGB + 红外/深度）
- **多级特征融合**: 在多个尺度进行特征拼接和增强
- **高性能推理**: 保持 Ultralytics 的高效推理速度
- **全系列支持**: YOLOv8、YOLO11、YOLO12、YOLOv26 双模态版本

## 📁 模型配置

### YOLO 双模态全系列

## ⚡ 快速使用

### 训练示例

```python
from ultralytics import YOLO

# 加载双模态模型（以YOLOv26为例）
model = YOLO("ultralytics/cfg/models/double_26/yolo26_double_backbone_fuse.yaml")

# 开始训练
results = model.train(
    data="ultralytics/cfg/datasets/VOC_LLVIP_double.yaml",
    epochs=100,
    imgsz=640,
    batch=16,
    device=[0, 1]  # 支持多GPU
)
```

### 数据集格式

双模态数据集需要包含两个图像目录：

```yaml
# VOC_LLVIP_double.yaml
path: /path/to/dataset/
train: 
  - images/train      # RGB图像
train2: 
  - images/train2     # 红外/深度图像
val: 
  - images/val
val2: 
  - images/val2

names:
  0: person
```

## ⚠️ 重要说明

标准 Ultralytics 库**不原生支持**双 backbone 配置，必须使用本项目修改版本：
- 修改了 `ultralytics/nn/tasks.py` 的 `parse_model` 函数支持 `backbone2`
- 扩展了 `BaseModel` 的 `_predict_once` 方法支持双输入
- 实现了双模态数据集加载和处理逻辑
